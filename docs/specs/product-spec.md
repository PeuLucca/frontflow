# Product Spec - Front Row

## One-liner

Front Row is a fast fashion-agency draft game where the player builds a celebrity lineup and beats a rival agency across major fashion/media events.

## Target User

People who enjoy:
- celebrity culture
- football WAGs
- fashion events
- model/influencer rankings
- simple viral browser games

## Main Inspiration

7a0:
- fast rounds
- simple choices
- hidden strategy
- replayability
- clear win/loss

## Theme

A glamorous agency rivalry inspired by:
- fashion week
- celebrity media
- luxury campaigns
- The Devil Wears Prada
- Anne Hathaway vs Meryl Streep type rivalry

## MVP Goal

Create a complete playable season:

- Home screen
- Agency intro
- Rival intro
- Draft with images
- 11 player selections
- Rival auto-draft
- Season simulation
- Event-by-event matchups
- Final trophy screen
- Restart button

## Victory Condition

The player wins if their agency beats the rival agency across the season events.

No abstract final score should be the main result.

## Core Feeling

The user should think:

- "Só mais uma partida."
- "E se eu tivesse escolhido outra?"
- "Essa rival me venceu, vou jogar de novo."
- "Meu elenco ficou absurdo."
