# UI Spec

## Design Goals

Front Row must feel:

- feminine
- luxury
- editorial
- glamorous
- competitive

## Inspiration

Keep from 7a0:

- centered layout
- simple flow
- strong card selection
- fast gameplay

Do NOT copy the visual design.

## Character Card

Must contain:

- photo
- name
- category
- choose button

Must NOT contain:

- hidden stats
